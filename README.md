# Hello World

A simple Python application that prints "Hello, World!".

## Running the Application

```bash
python hello.py
```

## Running Tests

```bash
python -m pytest test_hello.py -v
```

Or with unittest:

```bash
python -m unittest test_hello.py -v
```

## GitHub Actions

This repository includes automated tests that run on every push and pull request to verify the application output.
